Welcome to Random User Generator

  • @GET-https://assignment-randomuser-generator.vercel.app /user/random : to get a random user
  • @GET-https://assignment-randomuser-generator.vercel.app /user/all?limit=2 - to get a limited number of user
  • @GET- https://assignment-randomuser-generator.vercel.app /user/all - to get all the users
  • @POST-https://assignment-randomuser-generator.vercel.app /user/save : to save a user in db
  • @DELETE-https://assignment-randomuser-generator.vercel.app /user/delete/${id} - to delete a user
  • @PATCH- https://assignment-randomuser-generator.vercel.app /user/update/${id} - to update a user
